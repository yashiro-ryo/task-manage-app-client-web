# Task管理アプリ

カンバン方式でタスクを管理できるwebアプリケーションです。

現時点では以下の機能を実装しています。

- タスクグループ作成
- タスクグループ削除
- タスク作成
- タスク削除

## インストール方法

リポジトリのクローン

```git clone https://github.com/yashiro-ryo/task-manage-app-client-web```

必要なライブラリのインストール

```yarn```

開発モードでの起動

```yarn start```

Build

```yarn build```

Lint

```yarn lint-fix```
